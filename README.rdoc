== README

This is a demo app that uses the <a href="http://rubyonrails.org/" target="_blank">Ruby on Rails</a> web framework as a backend, a <a href="http://www.postgresql.org/">PostgreSQL</a> database, <a href="https://facebook.github.io/react/" target="_blank">React.js</a> for the data entry UI (Hours), and <a href="http://d3js.org/" target="_blank">D3.js</a> for data visualisations (Chart panel).
